# Web Store 스크린샷 캡션 최종본

권장 업로드: 5장 (1280x800 또는 1280x720)

## Screenshot 1 — Reader 모드 메인 화면
- **제목**: 기사 본문을 Reader 모드로 깔끔하게
- **캡션**: 광고/사이드바를 줄이고 본문 중심으로 재렌더링해 읽기 흐름을 유지합니다.
- **촬영 팁**: 좌측 상단에 기사 제목, 본문 단락 3~4개가 보이게 캡처

## Screenshot 2 — 문단 아래 한국어 번역
- **제목**: 원문 바로 아래 문단 번역 삽입
- **캡션**: 각 단락에 1:1로 한국어 번역을 붙여 원문과 번역을 함께 비교하며 읽을 수 있습니다.
- **촬영 팁**: 원문(검정) + 번역(파란 박스) 대비가 잘 보이게 확대

## Screenshot 3 — 설정 화면(API Key / OAuth)
- **제목**: 인증 방식 선택과 모델 설정
- **캡션**: API Key 또는 OAuth 방식으로 연결하고 모델/엔드포인트를 유연하게 설정할 수 있습니다.
- **촬영 팁**: Provider, Auth Type, 연결 테스트 버튼이 함께 보이게 캡처

## Screenshot 4 — 제외 도메인 보안 설정
- **제목**: 민감 도메인 번역 차단
- **캡션**: 메일·금융·사내 페이지 등 민감한 사이트는 제외 도메인으로 등록해 번역을 자동 차단할 수 있습니다.
- **촬영 팁**: 제외 도메인 입력란과 예시 값 노출

## Screenshot 5 — 팝업/단축키 사용성
- **제목**: 한 번의 클릭 또는 단축키로 즉시 번역
- **캡션**: 팝업에서 번역 시작/중지, 단축키로 빠른 토글이 가능합니다.
- **촬영 팁**: 브라우저 툴바 팝업 UI와 기사 배경을 같이 담기
