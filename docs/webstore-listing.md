# Web Store Listing 템플릿

## Short description
Reader mode Korean translation for English/Chinese articles with paragraph-by-paragraph insertion.

## Full description
LLM Translator Render helps you read English/Chinese web articles in Korean.

- Re-renders the page in Reader mode for stable layout
- Inserts Korean translation right below each original paragraph
- Supports API key and OAuth authentication flows
- Preserves context with chunk translation + glossary
- Includes retry queue and paragraph cache for faster reloads
- Optional inline mode and excluded-domain safety settings

Privacy notice:
Article text may be sent to your configured LLM provider for translation.
No analytics tracker is included in this extension.
